# SIWR - Ćwiczenie 2: Create with VR

Projekt VR w Unity 6 realizowany w ramach przedmiotu SIWR, oparty na kursie [Create with VR](https://learn.unity.com/course/create-with-vr) z Unity Learn.

## Wymagania

- Unity 6 (6.0+)
- XR Interaction Toolkit
- OpenXR Plugin
- Universal Render Pipeline (URP)
